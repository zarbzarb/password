# 密码管理器 Web应用需求文档

## 1. 项目概述

### 1.1 项目名称
密码管理器（Password Manager）

### 1.2 项目简介
一个轻量级的Web应用，用于安全存储和管理用户常用网站的账号密码信息。采用云数据库存储方式，支持多设备同步访问，支持移动端访问。

### 1.3 目标用户
需要管理多个网站账号密码的个人用户，希望在多设备间同步数据

---

## 2. 技术栈

### 2.1 前端框架
- **Vue 3** - 采用Composition API
- **Vite** - 构建工具

### 2.2 UI框架
- **Vant 4** 或 **Element Plus** - 移动端优先的UI组件库

### 2.3 数据存储
- **Supabase** - 云数据库（PostgreSQL），用于持久化存储密码数据
- **LocalStorage** - 本地浏览器存储，缓存加密后的密码数据
- **SessionStorage** - 会话存储，保存登录会话信息
- **数据导入/导出**：支持 JSON、Markdown、TXT、CSV 格式的导出，支持 JSON 格式的导入

### 2.4 后端服务
- **Supabase REST API** - 无需自建后端，直接使用 Supabase 提供的 REST API
- **Supabase Auth** - 用户认证服务
- **Supabase Row Level Security** - 行级数据隔离

### 2.5 其他
- **Vue Router** - 路由管理
- **Pinia** - 状态管理
- **Crypto-js** - 密码加密
- **@supabase/supabase-js** - Supabase SDK

---

## 3. 功能需求

### 3.1 核心功能

#### 3.1.1 密码条目管理
- **添加密码**：录入网站名称、网址、账号、密码、备注等信息
- **编辑密码**：修改已有密码条目
- **删除密码**：删除不需要的密码条目
- **查看密码**：点击查看详情，密码默认隐藏，点击可显示明文
- **复制功能**：一键复制账号/密码到剪贴板

#### 3.1.2 密码搜索
- 支持按网站名称搜索
- 支持模糊匹配

#### 3.1.3 密码分类
- 支持自定义分类（如：社交、工作、金融等）
- 支持按分类筛选查看

#### 3.1.4 数据安全
- 主密码保护：首次使用设置主密码
- 数据加密存储：使用AES-256加密算法
- 自动锁定：一段时间无操作自动锁定
- 会话保持：刷新页面后保持登录状态，无需重新输入主密码

### 3.2 辅助功能

#### 3.2.1 数据备份与恢复
- 多格式导出：支持导出为 JSON、Markdown、TXT、CSV 格式
- 明文导出：可直接导出明文密码，方便查看和使用
- 数据导入：支持导入 JSON 格式的密码数据
- Edge 密码导入：支持将 Microsoft Edge 浏览器导出的 CSV 密码文件转换为系统可导入的格式

#### 3.2.2 密码生成器
- 生成随机强密码
- 可自定义长度、字符类型

#### 3.2.3 密码强度检测
- 实时检测密码强度
- 给出安全建议

#### 3.2.4 网址前缀补全
- 添加密码时自动补全网站前缀（https://、http://、mailto: 等）
- 支持下拉选择常用前缀

## 6. 安全设计

### 6.1 加密方案
- 使用AES-256加密存储密码
- 主密码使用PBKDF2或bcrypt进行哈希存储
- 加密密钥由主密码派生

### 6.2 安全措施
- 使用 Supabase Row Level Security 实现用户级数据隔离
- 所有密码数据在前端加密后再上传到云端
- 主密码不存储在云端，仅在本地验证
- 支持 HTTPS 加密传输
- 支持清除所有数据功能

## 10. 注意事项

1. **数据安全**：所有敏感数据必须加密存储
2. **用户体验**：移动端操作要流畅便捷
3. **数据备份**：提醒用户定期备份数据
4. **浏览器限制**：LocalStorage有容量限制（通常5-10MB）
5. **隐私提示**：建议用户不要在公共电脑上保存敏感密码
