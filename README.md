# Password Manager

一个基于 Vue 3 的本地密码管理器，采用白色极简风格设计，专注于安全存储和便捷管理您的密码。

## 功能特性

### 核心功能
- **密码存储**：使用 AES-256 加密算法保护您的密码数据
- **主密码验证**：通过 SHA-256 双重哈希验证主密码，确保安全
- **分类管理**：支持按类别（社交、工作、金融、购物、学习、开发、其他）管理密码
- **搜索过滤**：快速搜索和过滤密码条目
- **自动锁定**：支持自动锁定功能，保护隐私

### 数据导入/导出
- **多格式导出**：支持导出为 JSON、Markdown、TXT、CSV 格式
- **明文导出**：可直接导出明文密码，方便查看和使用
- **数据导入**：支持导入 JSON 格式的密码数据
- **Edge 密码导入**：支持将 Microsoft Edge 浏览器导出的 CSV 密码文件转换为系统可导入的格式

### 用户体验
- **白色极简风格**：简洁美观的界面设计
- **网址前缀补全**：添加密码时自动补全网站前缀（https、http、mailto 等）
- **会话保持**：刷新页面后保持登录状态，无需重新输入主密码
- **响应式设计**：适配桌面和移动设备

## 技术栈

- **前端框架**：Vue 3 + Composition API
- **构建工具**：Vite
- **UI 组件库**：Vant 4
- **状态管理**：Pinia
- **路由管理**：Vue Router（Hash 模式）
- **数据存储**：LocalStorage + SessionStorage
- **加密技术**：AES-256 + SHA-256

## 使用说明

### 首次使用
1. 打开应用后，设置一个主密码（至少 4 位）
2. 记住您的主密码，它是访问所有密码的唯一凭证

### 添加密码
1. 点击页面底部的 "+" 按钮
2. 填写网站名称、网址、用户名、密码等信息
3. 选择分类，添加备注（可选）
4. 点击保存

### 导入密码
1. 点击页面右上角的设置图标
2. 选择 "导入数据"
3. 选择准备好的 JSON 文件（如 `Edge密码导入.json`）

### 导出密码
1. 点击页面右上角的设置图标
2. 选择 "导出数据"
3. 选择导出格式（JSON/Markdown/TXT/CSV）

### Edge 密码转换
1. 在 Edge 浏览器中导出密码为 CSV 文件
2. 使用系统提供的转换工具将 CSV 转换为 JSON 格式
3. 导入转换后的 JSON 文件

## 安全说明

- 所有密码数据存储在浏览器的 LocalStorage 中
- 数据使用 AES-256 加密，主密码使用 SHA-256 哈希存储
- 建议定期导出数据备份
- 请妥善保管您的主密码，丢失后无法找回
